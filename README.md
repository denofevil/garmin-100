# TypeScript Chrome Extension

This project is a Chrome extension developed using TypeScript, Webpack, React, and Redux. It includes a background script, a content script, and a popup interface.

## Project Structure

  - `background/` - Contains the background script.
  - `content/` - Contains the content script.
  - `popup/` - Contains the popup interface files.
  - `styles/` - Global CSS styles.
  - `assets/` - Images and icons.

## Features

- **100m Split Aggregation**: Automatically aggregates swim lengths into 100m/yard splits for easier analysis
- **Fastest/Slowest Highlighting**: The fastest split is highlighted in green, and the slowest split is highlighted in red for quick visual comparison
